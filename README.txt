==================================================
✨ 少儿英语趣味朗读打分系统 (English Spark!)
==================================================

这是一个完全纯前端构建、开箱即用的少儿英语 AI 语音评测模拟系统。

📦 压缩包内容说明：
1. index.html   - 包含主界面、Tailwind CSS样式、交互算法及语音对比系统的核心文件。
2. README.txt   - 本说明文件。

🚀 快速开始运行：
1. 解压该 ZIP 压缩包到你的本地目录。
2. 双击运行 "index.html" 即可在任意现代浏览器中完美呈现。
3. 麦克风权限说明：
   - 浏览器原生语音识别 (Web Speech API) 安全机制要求：
     在本地 file:// 协议直接双击打开可能无法授权麦克风；
     推荐将 index.html 部署在你的网页服务器(如 Nginx/Apache/Node.js) 或通过本地 localhost 服务运行；
   - 💡 特别设计：如果你无法使用麦克风，我们在应用中内置了【完美/良好/一般】三档发音“模拟打分器”！双击即可一键模拟体验所有打分和星星墙排行逻辑！

感谢您的下载与使用！如有修改，直接编辑 index.html 即可。
